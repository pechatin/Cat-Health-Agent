# Ветеринарное Руководство: Анализ Препаратов и Комплаенса

Этот документ предназначен для критического анализа ветеринарных назначений и оценки возможности их применения у Пациента.

## 1. КАТЕГОРИИ ПРЕПАРАТОВ (Кратко)

### 1.1. Антибиотики
*   **Маркеры**: Амоксиклав (Синулокс), Марфлоксицин, Энрофлоксацин.
*   **Критика**: Требуйте подтверждения инфекции (WBC↑, палочки ↑, посев). «Для профилактики» — повод для вопроса врачу.
*   **Побочки**: Диарея, рвота (у кошек часто от горечи).

### 1.2. Почечные препараты (ХПН)
*   **Маркеры**: Семинтра (Телмисартан), Фортекор (Беназеприл), Ипакитине/Реналцин (сорбенты).
*   **Логика**: Семинтра снижает давление и протеинурию (белок в моче). Фосфат-биндеры (сорбенты) — если фосфор выше 1.5. 
*   **Критика**: Если креатинин 1000, семинтра может быть фатальной (резкое падение фильтрации). Проверяйте стадию и КЭК (калий, фосфор).

### 1.3. При стимуляции аппетита
*   **Маркеры**: Миртазапин (Миртаза), Капиморелин (Энтурайс).
*   **Комплаенс**: Миртазапин в виде трансдермального геля (на ухо) — спасение для кошек, которых нельзя кормить таблетками.

## 2. ФАКТОР КОМПЛАЕНСА (Как коту давать лекарства?)

Невозможность дать препарат — это медицинский риск, а не просто «каприз». 
**Алгоритм действий агента:**
1.  **Ранжирование**: Разделяйте назначения на группы:
    *   **Tier 1 (Жизненно важные)**: Антибиотики (курс нельзя прерывать), Гормоны, Препараты от давления.
    *   **Tier 2 (Поддерживающие)**: Витамины, Гепатопротекторы, БАДы. Можно пропустить, если Кот в стрессе.
2.  **Замена формы**: Если Кот плюется таблетками — ищите тот же препарат в виде сиропа, инъекций или трансдермального геля.

## 3. ПОИСК ИЗЪЯНОВ В НАЗНАЧЕНИЯХ

Твоя задача — задавать вопросы к схеме лечения:
*   «Назначено 3 гомеопатических препарата. Зачем? Какое действие они оказывают при ХПН 3 стадии?»
*   «Препарат Х токсичен для печени, а у Пациента АЛТ на верхней границе. Есть ли альтернатива?»
*   «Врач назначил физраствор (0.9% NaCl) подкожно, но у Кота застойная сердечная недостаточность. Это риск отека легких!»

## 4. АНТРОПОМОРФИЗМ (Двойной контроль)

При анализе дневника Тараса ты должен искать скрытые симптомы за словами-интерпретациями:
*   "Ему невкусно" -> "Возможна тошнота (уремический гастрит) или язвы во рту". Посмотри на мочевину.
*   "Он ленится" -> "Летаргия, слабость из-за анемии (HCT↓)".
*   "Обиделся" -> "Избегание контактов может быть признаком хронической боли".
